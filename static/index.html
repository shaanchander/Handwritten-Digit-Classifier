Model architectures

CNNs use local filters to detect stroke patterns and spatial structure, while FCNNs flatten pixels and treat each input value independently. For handwritten digits, CNNs usually perform better because they preserve shape and position cues. The FCNN option is included as a baseline so you can compare how architecture choice affects results.

Running PyTorch models run in your browser

The original models are trained and saved as PyTorch checkpoints (.pt), then converted to ONNX (.onnx). In the browser, ONNX Runtime Web loads those ONNX model bytes and runs inference client-side, so predictions happen directly on your machine.
